# SQL tricks
SQL tricks are a set of useful SQL scripts everybody supporting MS SQL DBs should have at hand:
* update statistics
* review an execution plan
* rebuild indexes
* autofix a DB user
* disable / enable table triggers
* disable / enable DB integrity check
